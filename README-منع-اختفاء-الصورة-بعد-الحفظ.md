# منع اختفاء الصورة بعد الحفظ

استبدل الملف:

```text
js/admin.js
```

يعالج هذا التعديل حالتين:

1. عند اختيار صورة من الجهاز تبقى المعاينة ظاهرة بعد الضغط على حفظ أثناء الجلسة الحالية.
2. إذا كانت خانة الصورة فارغة عند الحفظ لا يتم مسح مسار الصورة السابق من بيانات العنصر.

ملاحظة مهمة: للظهور الدائم بعد إغلاق الصفحة أو فتح الموقع من جهاز آخر، يجب أن يكون ملف الصورة موجودًا فعليًا داخل مجلد images المناسب، وأن يكون المسار محفوظًا في JSON/JS.
