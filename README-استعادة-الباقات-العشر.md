# استعادة الباقات العشر ومنع طمسها بالحفظ المؤقت

استبدل هذه الملفات داخل مشروعك:

```text
js/data-service.js
data/research-dynamic-data.json
data/research-dynamic-data.js
```

## سبب المشكلة

كانت لوحة التحكم تقرأ أحيانًا نسخة محفوظة في المتصفح `localStorage` تحتوي على باقة واحدة فقط. لذلك حتى بعد استبدال ملفات `data`، بقيت القائمة تعرض باقة الباحث العلمي فقط.

## ما الذي يعالجه هذا التعديل؟

- إذا كان الحفظ المؤقت يحتوي باقات أقل من ملف المشروع، يتم تجاهله تلقائيًا.
- يتم تحميل ملف العشر باقات من `data/research-dynamic-data.js` أو `data/research-dynamic-data.json`.
- صفحات الزائر تقرأ ملفات المشروع فقط ولا تتأثر بالحفظ المؤقت.

## بعد الاستبدال

افتح `admin.html` ثم اضغط `Ctrl + F5`.

إذا بقيت باقة واحدة ظاهرة، اضغط زر `استعادة الأصل` مرة واحدة، ثم اضغط `Ctrl + F5`.
