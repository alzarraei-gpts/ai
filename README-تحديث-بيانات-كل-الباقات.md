# تحديث بيانات كل الباقات

هذا التحديث يستبدل بيانات التقسيمات والبوتات اعتمادًا على ملف: `كل الباقات.docm`.

## طريقة التركيب

استبدل الملفين داخل مشروعك:

```text
data/research-dynamic-data.json
data/research-dynamic-data.js
```

بعد الاستبدال افتح `index.html` واضغط `Ctrl + F5`.
