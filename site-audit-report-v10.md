# تقرير فحص وتقوية ملفات الواجهة

رقم الإصدار: `2026-06-30-01`

## ملخص العمل
- إعادة بناء ملفات HTML بصورة أنظف وأخف، وإزالة التراكمات والحقن السابقة.
- تثبيت زر `نسخ الوصف` داخل نافذة الوصف في الجهة اليسرى المقابلة لموضع الإغلاق، مع إطار وخلفية متسقة.
- إضافة زر `المساعد الذكي` فوق زر `الدعم المباشر` في أسفل الجهة اليسرى، مع لوحة بحث خفيفة داخلية عن البوتات والروابط.
- إبقاء زر واتساب للدعم المباشر دون عرض الرقم في الواجهة.
- ترتيب الشريط السفلي بروابط التواصل وأيقونات خفيفة محلية دون مكتبات خارجية.
- تفعيل `defer` لملفات JavaScript المحلية، وإضافة lazy/async للصور عند التشغيل.
