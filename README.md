# باقة الباحث العلمي — نسخة لوحة التحكم الموحّدة

هذه النسخة تفصل الواجهة عن لوحة التحكم، وتوحّد طريقة إدراج الصور والنصوص في ثلاثة مستويات:

1. الباقة: صورة عرضية 16:9 + عنوان + وصف.
2. التقسيم: صورة مربعة 1:1 + عنوان + وصف.
3. البوت: صورة مربعة 1:1 + عنوان + وصف + حدود + مثال + روابط ChatGPT/Gemini.

## طريقة التشغيل

افتح `admin.html` لإدارة المحتوى، وافتح `index.html` للواجهة العامة.

## طريقة الحفظ الدائم

- ضع الصور داخل مجلدات `images/packages` و `images/sections` و `images/bots`.
- استخدم المسار الدائم المقترح داخل لوحة التحكم.
- اضغط حفظ، ثم صدّر JSON.
- استبدل `data/research-dynamic-data.json` بالملف الجديد.

## ملاحظة

اختيار صورة مؤقتة من الجهاز يدمج الصورة داخل JSON، وهو مناسب للتجربة لا للنشر النهائي إذا كانت الصور كثيرة.


## تشغيل لوحة التحكم بطريقة استيراد JSON

إذا فتحت `admin.html` وظهرت رسالة تعذر قراءة البيانات، فهذا غالبًا بسبب فتح المشروع محليًا من المتصفح. الحل المعتمد في هذه النسخة:

1. اضغط زر **استيراد JSON للبدء**.
2. اختر الملف: `data/research-dynamic-data.json`.
3. ستظهر الباقة والتقسيمات والبوتات داخل لوحة التحكم.
4. بعد أي تعديل اضغط حفظ، ثم **تصدير JSON**.
5. استبدل الملف القديم داخل `data/research-dynamic-data.json` بالملف المصدّر.

هذه الطريقة لا تحتاج تشغيل سيرفر محلي، وتبقي الواجهة مفصولة ومنظمة.


## إصلاح الاستيراد
- زر الاستيراد يقبل الآن `research-dynamic-data.json`، كما يقبل ملف `research-dynamic-data.js` عند الحاجة.
- إذا تم اختيار `image-map.json` سيظهر تنبيه يوضح أنه ملف خريطة صور وليس ملف بيانات.
- لا تستخدم `image-map.json` لتشغيل لوحة التحكم.


## تحديث سياسة الصور

هذه النسخة لا تفرض أسماء ثابتة للصور. يمكنك وضع الصورة داخل مجلدها المناسب بالاسم الأصلي، ثم حفظ المسار النسبي في لوحة التحكم:

- `images/packages/اسم-الصورة`
- `images/sections/اسم-الصورة`
- `images/bots/اسم-الصورة`

استخدم زر **اختيار صورة باسمها الأصلي** لتعبئة المسار بسرعة، ثم انسخ الصورة يدويًا إلى المجلد المناسب، وبعدها اضغط **حفظ** ثم **تصدير JSON**.


## إصلاح عرض الصور في التقسيمات والبوتات
- زر "اختيار صورة باسمها الأصلي" يحفظ الآن مسار الصورة تلقائيًا داخل بيانات العنصر الحالي.
- زر "اعتماد المسار الحالي" يحفظ المسار داخل البيانات مباشرة، وليس مجرد معاينة.
- زر "اختبار الظهور" يتحقق من أن الصورة موجودة فعلًا في المسار المكتوب.
- عند تصدير JSON يجب استبدال `data/research-dynamic-data.json` بالملف الجديد، ثم تحديث الصفحة بقوة `Ctrl + F5`.
