# تعديل ثبات الصورة بعد الحفظ والتحديث

استبدل الملف:

```text
js/admin.js
```

يعالج هذا التعديل اختفاء الصورة بعد الضغط على أزرار الحفظ، وذلك عبر:

- عدم إعادة رسم بطاقة التحرير عند الحفظ العادي.
- حفظ معاينة الصورة محليًا في المتصفح عند اختيارها.
- إبقاء المعاينة ظاهرة بعد الحفظ والتحديث المحلي.
- إبقاء شرط النشر النهائي: يجب وضع الصورة نفسها داخل مجلد images المناسب.

بعد الاستبدال افتح `admin.html` واضغط `Ctrl + F5`.
