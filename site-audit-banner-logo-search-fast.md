# تقرير ضبط البانر والشعار ومحاذاة البحث — 2026-07-01-banner-logo-search-fast

## نطاق العمل
- اعتماد البانر المرفق بوصفه الصورة الحالية داخل `images/discount-80-banner.png`.
- تحريك الشعار في الترويسة قليلًا جهة اليسار حتى لا يلتصق بحافة الشاشة اليمنى أو يختفي جزء منه.
- إعادة ضبط مربع البحث `ابحث هنا...` ليأخذ عرض الحاوية نفسها ولا يخرج عن محاذاة بقية الاسكتشات.
- تثبيت المؤثرات على البانر: حركة خفيفة، لمعان عابر، هالات خضراء/ذهبية، وخط إضاءة سفلي.
- تحسين الإقلاع على الجوال عبر تثبيت أبعاد البانر، ضغط الصورة، تقليل الاعتماد على خصائص ثقيلة، وتخفيف تكرار تحديث البحث.

## فحص الفجوات
- لا توجد روابط خارجية غير آمنة داخل أزرار التشغيل؛ الروابط الخارجية تمر عبر `safeUrl` وتفتح بـ `noopener noreferrer`.
- تم ضبط `CSP` في الصفحات الأساسية.
- الصور المفقودة لا تكسر التخطيط؛ يتم التعامل معها داخل `app.js`.
- مربع البحث أصبح داخل `toolbar shell` بعرض مضبوط ومتسق.
- إغلاق النوافذ يتم بزر الإغلاق، الضغط خارج النافذة، وزر `Escape`.

## ملاحظة تشغيلية
للحصول على أفضل نتيجة في GitHub Pages، ارفع المجلدات كاملة: `css`, `js`, `data`, `images` مع ملفات HTML الأربعة.
