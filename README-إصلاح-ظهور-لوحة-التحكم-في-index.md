# إصلاح ظهور لوحة التحكم داخل index.html

استبدل الملفات التالية فقط داخل مشروعك:

- index.html
- package.html
- bots.html
- js/app.js

الغرض من التعديل:

- جعل index.html واجهة زائر عامة فقط.
- منع ظهور لوحة التحكم داخل الصفحة الرئيسية.
- إبقاء لوحة التحكم في admin.html فقط.
- توحيد مسار الزائر: الباقات ← التقسيمات ← البوتات.

بعد الاستبدال افتح index.html واضغط Ctrl + F5.
